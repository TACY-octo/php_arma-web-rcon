# Interface WEB ARMA 3 RCON 

## Besoin 

Pouvoir administrer un serveur ARMA 3 à partir d'une interface WEB

## Fonctionnement

L'utilisateur se connect sur l'interface web, renseigne un login et mot de passe. Une fois connecté, il à la possibilité de modifier les informations RCon (IP du serveur, Ports et mot de passe RCON).

## Installation 

Dans un premier temps, vous devez configurer le fichier suivant : "database.php" ce fichier permet de connecter la base de doonées au site web php. 

Vous devez ensuite exécuter le script "init_database.sql" afin de créer les tables pour le bon fonctionnement du site.
